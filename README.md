## Portfolio

Projeto de portfólio minimalista com paleta tipo Gruvbox, tipografia confortável e seções de Sobre, Projetos e Contato.

### Como rodar
- Abra o `index.html` diretamente no navegador ou use uma extensão de servidor local (Live Server).

### Estrutura
- `index.html`: marcação principal
- `styles.css`: estilos (paleta Gruvbox, layout responsivo, tipografia)
- `script.js`: pequenas interações (menu mobile, tema claro/escuro)

### Personalização
- Atualize seus dados em `index.html` nas seções Sobre, Projetos e Contato.
- Ajuste a paleta em `styles.css` alterando as variáveis CSS em `:root`.
